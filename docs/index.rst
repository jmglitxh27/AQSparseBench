AQSparseBench
=============

**AQSparseBench** builds sparse air-quality monitoring network benchmarks from EPA AQS (and pluggable sources), scores and generates candidate networks, clusters representatives, and exports Parquet bundles for transfer-learning experiments.

Source repository: https://github.com/jmglitxh27/AQSparseBench
